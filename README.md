# 仿《战神》战斗系统实现

## 现阶段完成情况

- 实现了英雄角色的移动与视野移动能力
- 实现了斧头武器，并将其附着在英雄角色背部
- 通过GAS（Gameplay Ability System）实现斧头武器的激活能力



## 开发环境

- 操作系统：Windows11（64位）
- UE版本：5.6
- 主要编程语言：C++
